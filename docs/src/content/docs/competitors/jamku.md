## Overview

| Attribute | Detail |
|---|---|
| **Founded** | 2013 |
| **HQ** | India |
| **Pricing** | Free <3 users, Rs 150+/user/month |
| **Free Tier** | Yes (up to 3 users) |
| **Users** | 10,000+ |
| **Funding** | Bootstrapped |
| **Key Differentiator** | Oldest player, free tier, IT+GST auto-fetch |

Jamku is the **oldest dedicated CA practice management tool** in India, operating since 2013. Its longevity and free tier for small teams have built a substantial user base of 10,000+ users, making it the largest bootstrapped PMS by install count.

## Key Features

- **Task Management:** Comprehensive task tracking with recurring workflows
- **IT Portal Auto-Fetch:** Automated data retrieval from Income Tax portal
- **GST Portal Fetch:** Automated data retrieval from GST portal
- **Client Management:** Client master with credentials and service mapping
- **Billing:** Basic invoicing and outstanding tracking
- **Tally Integration:** Export/import with Tally accounting
- **Mobile App:** Basic mobile access
- **Client Portal:** Simple client-facing status view

:::note[Portal Fetch Advantage]
Jamku's IT and GST portal auto-fetch is a genuine differentiator -- only one of two PMS tools (along with partial support in Finexo) that offers this. This feature alone keeps many CAs locked in.
:::

## Strengths

1. **Free tier** -- 3 users free forever, strong for solo CAs and tiny firms
2. **IT + GST portal auto-fetch** -- saves hours of manual portal checking
3. **Largest installed base** -- 10K+ users, strong word-of-mouth
4. **Longevity** -- 10+ years of operation builds trust
5. **Tally integration** -- bidirectional data flow with Tally

## Weaknesses

| Area | Issue |
|---|---|
| **UI/UX** | Severely dated interface, no modern design language |
| **Tech Stack** | Aging architecture limits feature velocity |
| **WhatsApp** | No WhatsApp integration |
| **AI** | No AI capabilities |
| **Document Intelligence** | No OCR, extraction, or processing |
| **Client Portal** | Very basic, no branding or custom domain |
| **Analytics** | Limited reporting capabilities |
| **Innovation** | Slow release cadence, incremental updates only |

## CAOS Competitive Strategy

:::tip[How CAOS Wins Against Jamku]
- **Modern tech stack** and UX vs. decade-old interface
- **AI features** that Jamku cannot retrofit on its aging architecture
- **Document intelligence** as a new capability entirely
- **WhatsApp integration** for client communication
- **Multi-portal fetch** (IT + GST + MCA + TDS + PF) vs. Jamku's IT + GST only
- **Advanced analytics** vs. basic reports
:::

### Migration Path
Jamku users are sticky due to years of historical data. CAOS should offer **data import tools** that can ingest Jamku exports (CSV/Excel) and map them to CAOS entities. The free tier users are the easiest to convert -- offer a compelling free tier or trial that exceeds Jamku's limitations.

### Key Talking Points
- "Everything you love about Jamku's portal fetch, plus 4 more portals and AI-powered insights"
- "Your data deserves a modern home -- import your 10 years of Jamku data in one click"
- "WhatsApp and document intelligence are not features Jamku can add to a 2013 architecture"
