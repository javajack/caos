## Overview

| Attribute | Detail |
|---|---|

| **Pricing** | Rs 100/user/month (flat) |
| **Free Tier** | No |
| **Users** | 300+ CAs |
| **Funding** | Bootstrapped |
| **Key Differentiator** | Modern UX, flat pricing, geo-attendance, SOP management, Tally |

Turia is a **modern CA practice management tool** that stands out with clean UX, flat per-user pricing (no tiered feature gates), **geo-attendance tracking**, and **SOP (Standard Operating Procedure) management**. With 300+ CAs and Tally integration, it is building a solid foundation.

## Key Features

- **Task Management:** Clean task board with assignment and tracking
- **Geo-Attendance:** GPS-based attendance for field staff and client visits
- **SOP Management:** Create and manage standard operating procedures
- **Tally Integration:** Data sync with Tally accounting
- **Client Management:** Client master with service mapping
- **Billing:** Basic invoicing and fee management
- **Compliance Calendar:** Deadline tracking for major compliances
- **Team Management:** Staff workload, attendance, and performance
- **Mobile App:** Mobile access with geo-attendance support

:::note[Modern and Growing]
Turia is one of the few PMS tools with a genuinely modern interface. While its user base is small (300+ CAs), its design approach and feature choices show product thinking that most competitors lack.
:::

## Strengths

1. **Modern UX** -- clean, contemporary interface
2. **Flat pricing** -- Rs 100/user, no feature gates or tiers
3. **Geo-attendance** -- unique GPS-based attendance for field teams
4. **SOP management** -- standardize firm processes
5. **Tally integration** -- essential connectivity

## Weaknesses

| Area | Issue |
|---|---|
| **Small user base** | Only 300+ CAs, limited market proof |
| **No WhatsApp** | No client communication integration |
| **No AI** | No AI-powered features |
| **No document intelligence** | No OCR or extraction |
| **No client portal** | No client-facing interface |
| **Limited compliance** | Basic calendar, no auto-fetch or filing |
| **Limited analytics** | Basic reporting only |

## CAOS Competitive Strategy

:::tip[How CAOS Wins Against Turia]
- **AI and WhatsApp** as major differentiators
- **Document intelligence** -- the capability gap Turia cannot fill easily
- **Client portal** for client self-service
- **Deeper compliance** with auto-fetch and 600+ workflows
- **Adopt geo-attendance** -- CAOS should include GPS attendance for field teams
- **Adopt SOP management** -- process standardization is valuable
:::

### Respect the Design
Turia's UX quality is a benchmark. CAOS must match or exceed this design standard -- competing on features alone against a tool with better UX is difficult. Design quality is a retention driver.
