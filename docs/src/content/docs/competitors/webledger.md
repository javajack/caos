WebLedger is a **cloud accounting platform with practice management features**, founded by a CA who understands practitioner pain points. With modest $270K funding and WhatsApp integration, it bridges the gap between pure accounting software and practice management tools.

## Key Features

- **Cloud Accounting:** Full double-entry bookkeeping in the cloud
- **Practice Management:** Basic task and client management for CA firms
- **WhatsApp Integration:** Client notifications and basic messaging
- **Billing:** GST-compliant invoicing
- **Multi-Client:** Manage multiple client books from one dashboard
- **Bank Reconciliation:** Automated bank statement matching
- **GST Compliance:** Basic GST return preparation
- **Reports:** Financial reports and practice dashboards

:::note[Accounting-First, PMS-Second]
WebLedger is primarily a cloud accounting tool with PMS features added on top. The accounting engine is the core product; practice management is a secondary layer. This makes it strong for CAs who do bookkeeping but weaker for compliance-heavy practices.
:::

## Strengths

1. **CA founder** -- built by a practitioner who understands CA workflows
2. **Accounting + PMS** -- unified bookkeeping and practice management
3. **WhatsApp** -- one of only 3 products with WhatsApp integration
4. **Affordable** -- Rs 3,999/year flat pricing
5. **Cloud-native** -- modern cloud architecture

## Weaknesses

| Area | Issue |
|---|---|
| **Shallow PMS** | Practice management features are basic |
| **Limited compliance** | No deep GST/ITR/TDS automation |
| **No Tally** | No Tally integration (positions as Tally replacement) |
| **No AI** | No AI features |
| **No document intelligence** | No OCR or extraction |
| **Small team** | $270K funding limits engineering capacity |
| **No client portal** | No client-facing interface |

## CAOS Competitive Strategy

:::tip[How CAOS Wins Against WebLedger]
- **Deeper PMS** with comprehensive compliance workflows
- **Document intelligence** as a core differentiator
- **AI-powered WhatsApp** vs. basic WhatsApp notifications
- **Multi-portal auto-fetch** for automated compliance data
- **Client portal** with self-service capabilities
- **Tally integration** (CAs will not abandon Tally for cloud accounting alone)
:::

WebLedger validates that CAs want **accounting + practice management** in one tool. CAOS should learn from this but lead with compliance and document intelligence rather than bookkeeping.
