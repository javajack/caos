## Overview

| Attribute | Detail |
|---|---|
| **Founded** | ~2022 |
| **HQ** | Bangalore, Karnataka |
| **Pricing** | Free (5 users), Rs 150/user/month |
| **Free Tier** | Yes (up to 5 users) |
| **Users** | Growing |
| **Funding** | ICAI-backed |
| **Key Differentiator** | ICAI backing, generous free tier, Bangalore tech ecosystem |

Papilio is an **ICAI-backed practice management tool** offering a generous free tier (5 users) and institutional credibility. Based in Bangalore, it benefits from access to India's tech talent pool but struggles with limited integrations and a narrow feature set.

## Key Features

- **Task Management:** Standard task assignment and tracking
- **Client Management:** Client master with service mapping
- **Billing & Invoicing:** GST-compliant invoicing
- **Compliance Calendar:** Basic deadline tracking
- **Client Portal:** Simple client-facing status view
- **Team Management:** Basic staff and workload management
- **Document Storage:** File storage and sharing with clients

:::note[ICAI Backing]
Papilio's ICAI backing provides institutional credibility and potential distribution advantages. However, unlike SimplifyPractice's active ICAI MOU, Papilio's relationship appears more passive -- recognition rather than active promotion.
:::

## Strengths

1. **ICAI backing** -- institutional credibility
2. **Free 5 users** -- most generous free tier for a PMS
3. **Bangalore base** -- access to strong engineering talent
4. **Simple to adopt** -- straightforward features, easy onboarding

## Weaknesses

| Area | Issue |
|---|---|
| **Limited integrations** | No Tally, no WhatsApp, no portal fetch |
| **No AI** | No AI-powered features |
| **No document intelligence** | No OCR or extraction |
| **No mobile app** | No dedicated mobile application |
| **Higher paid pricing** | Rs 150/user is above market average |
| **Basic features** | Feature set is shallow compared to QwikCA or SimplifyPractice |
| **Limited analytics** | Minimal reporting capabilities |

## Pricing Details

| Plan | Price | Details |
|---|---|---|
| Free | Rs 0 | Up to 5 users, core features |
| Paid | Rs 150/user/month | Beyond 5 users, all features |

## CAOS Competitive Strategy

:::tip[How CAOS Wins Against Papilio]
- **Deeper features** at every level -- compliance, billing, analytics
- **Integrations** -- Tally, WhatsApp, portal fetch are missing from Papilio
- **AI and document intelligence** as major differentiators
- **Mobile app** -- Papilio does not have one
- **Better value** -- more features at competitive pricing
- **Match the free tier** -- offer a compelling free or trial plan
:::

### Free Tier Competition
Papilio's 5-user free tier and Jamku's 3-user free tier set the floor for what CAs expect to try before paying. CAOS should offer a meaningful free tier or trial to compete for first adoption.
