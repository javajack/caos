:::tip[Layer Above Tally]
CAOS works on a layer above existing tools. If Tally is the engine, CAOS is the cockpit. We manage workflow, clients, communication, documents, billing, and intelligence -- we never replace the accounting backbone.
:::

## What CAOS Does NOT Do

| We Don't Do This | Why | Use This Instead |
|---|---|---|
| Accounting / bookkeeping | Tally handles this; we work above it | Tally, Zoho Books |
| GST return filing (initially) | Complex, requires GSP license | ClearTax, TaxOne |
| ITR preparation/filing | Government portal does this | IT e-filing portal |
| Payroll processing | Not core to practice management | Zoho Payroll, HROne |
| Full ERP | CAOS is lightweight PMS, not ERP | Tally, Busy |
| Legal research | Different product category | Vidur, Vaive AI |
